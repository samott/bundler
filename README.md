# ERC4337 Bundler Service

## Installation

```bash
$ npm install
```

## Configuration

Fill in `config/default.yaml`. The test user must be ready to handle
user ops for testing (e.g. have funds for gas).

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e
```
